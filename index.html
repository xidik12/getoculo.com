Features

Everything AI needs
to browse the web

Built from the ground up for AI agents. Five MCP tools, zero token waste, full security.

AI Chat Panel

Built-in AI sidebar with multi-provider support. Claude, OpenAI, Gemini, Grok, and OpenClaw. Chat about any page, summarize articles, extract data — all without leaving the browser.

5 MCP Tools

page, act, fill, read, run — five tools is all Claude Code needs. Page descriptions in ~30 tokens. 99% fewer tokens than DOM snapshots.

Smart Passkey & Auth

Credential vault backed by macOS Keychain with AES-256 encryption. AI says “login to Gmail” — Oculo fills credentials securely. Passwords never touch the AI.

Media Generation

Generate images with Nano Banana and videos with Veo 3.1 directly from the browser. AI-powered creative tools integrated into your browsing workflow.

Privacy & Security

Four-tier permission gates (auto/notify/confirm/blocked). PII redaction on every MCP response. Anti-prompt-injection. Encrypted audit log. Touch ID for sensitive actions.

Voice Control

On-device Whisper speech-to-text. Talk to the AI chat panel or dictate form inputs. Also powers autonomous audio CAPTCHA solving — zero cloud calls, zero tokens.

How It Works

Three steps to AI browsing

From install to AI-powered web interaction in under a minute.

01

Install Oculo

Download the macOS app or install via npm. Launch it and Oculo starts as both a browser and an MCP server.

npm install -g oculo
02

Connect Claude Code

Register Oculo as an MCP server in Claude Code with one command. The bridge connects via a local HTTP channel — secure and fast.

claude mcp add oculo ...
03

AI Browses For You

Ask Claude to navigate, fill forms, extract data, or run multi-step workflows. Oculo describes pages in ~30 tokens — 99% less than DOM snapshots.

“Search for flights to Tokyo”

MCP Integration

Connect in one command

Register Oculo as an MCP server and unlock 5 powerful tools for Claude Code.

Terminal 
# Register Oculo as an MCP server for Claude Code
$ claude mcp add oculo -- node ~/Desktop/oculo/bin/oculo-mcp.mjs
page ~30 tok

Describe current page — headings, forms, buttons, links. Compact summaries, not DOM dumps.

act ~10 tok

Navigate, click, scroll, press keys, login with vault credentials. Intent-based, not selector-based.

fill ~15 tok

Fill forms by label — name, email, country. Smart field matching with auto-submit option.

read ~100 tok

Extract structured data — search results, tables, lists, product details. Text or JSON output.

run ~200 tok

Multi-step pipelines — chain navigate, fill, click, wait, and read in a single tool call.

media varies

Generate images and videos via Nano Banana and Veo 3.1. Creative AI tools at your fingertips.

Example — Claude Code using Oculo 
# Claude Code prompt:
"Go to Hacker News and get the top 5 stories"

# Oculo handles it in ~3 tool calls:
act  → navigate to news.ycombinator.com        # 10 tokens
page → "Hacker News | 30 stories, nav, search"  # 30 tokens
read → extract top 5 with title, points, link  # 80 tokens

# Total: ~120 tokens (vs ~25,000 with DOM snapshots)
